Dental AI Platform - Project Progress Report
=========================================

I. Đã làm rồi (Completed)
-------------------------
1. Kiến trúc hệ thống:
   - Đã thiết kế kiến trúc microservices: Frontend (React), Gateway (Node.js/Express/Prisma), AI Service (Python/FastAPI), Database (Postgres), Storage (MinIO), Cache (Redis).
   - Đã cấu hình Docker Compose để chạy toàn bộ hệ thống.

2. Frontend:
   - Đã khởi tạo project React + TypeScript + Vite, cấu hình ESLint, TypeScript.
   - Đã có giao diện mẫu (App.tsx, main.tsx, index.html, CSS).
   - Đã cài đặt các package cơ bản: react, react-dom, react-router-dom, zustand, axios, @tanstack/react-query.

3. Gateway:
   - Đã khởi tạo project Node.js + TypeScript, cấu hình Prisma ORM, Dockerfile.
   - Đã có các route cơ bản: /uploads (mock), /analysis (tạo và lấy kết quả phân tích).
   - Đã có migration schema cho User, Image, Analysis.
   - Đã cấu hình .env, scripts Prisma migrate/generate.

4. AI Service:
   - Đã khởi tạo project Python + FastAPI, Dockerfile, requirements.txt.
   - Đã có route mẫu /debug-data-paths kiểm tra đường dẫn dữ liệu.
   - Đã cấu hình đọc biến môi trường DATA_ROOT, RAW_DIR, PROCESSED_DIR, SPLITS_DIR.

5. Scripts:
   - Đã có script Python để chuẩn bị, chia tách, xuất dữ liệu COCO, visualize dataset (download_and_prepare_dental_dataset.py).
   - Đã có requirements.txt cho scripts.

6. Data:
   - Đã tổ chức thư mục dữ liệu: raw, processed, splits, các dataset mẫu (children-panorama, ...).

7. Tài liệu:
   - Đã có README chi tiết về kiến trúc, cách cài đặt, sử dụng, troubleshooting, API endpoints.
   - Đã có .env.example cho các service.
